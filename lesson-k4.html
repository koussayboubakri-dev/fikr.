انشر مشروعك! 🚀

موقعك للعالم — GitHub Pages

GitHub Pages يوفر استضافة مجانية لأي موقع HTML — ٣ خطوات وموقعك حي على الإنترنت!

١
أنشئ حساب
اذهب إلى github.com وسجّل حساباً مجانياً بدقيقة واحدة
٢
ارفع ملفاتك
أنشئ مستودع جديد وارفع ملفات موقعك (index.html + CSS + JS)
٣
فعّل Pages
Settings → Pages → Source: main branch → موقعك حي على الإنترنت!

💡 بديل أسهل: Replit

اذهب إلى replit.com ← أنشئ Repl جديد من نوع HTML ← ارفع ملفاتك ← الموقع حي فوراً!
الرابط يكون: username.repl.co/project-name
