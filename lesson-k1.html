Bubble Sort — الفقاعات الذكية! 🫧

كيف يرتب الحاسوب قائمة أرقام؟ يقارن كل رقمين متجاورين ويبدّل موقعهما إذا كانا بترتيب خاطئ — مراراً حتى تُرتَّب القائمة!

📊 مرئية Bubble Sort التفاعلية
اضغط "خطوة" لبدء الخوارزمية...
bubble_sort.py
1def bubble_sort(قائمة):
2ن = len(قائمة)
3for i in range(ن):
4for j in range(0, ن-i-1):
5if قائمة[j] > قائمة[j+1]:
6قائمة[j], قائمة[j+1] = قائمة[j+1], قائمة[j]
7return قائمة
8 
9أرقام = [64, 34, 25, 12, 22]
10print(bubble_sort(أرقام))

List Comprehension — Python الذكي! ⚡

2مربعات = []

5
