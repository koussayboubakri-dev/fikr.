المفهوم الأساسي

HTML = الهيكل، CSS = الجمال! 🏗️🎨

كل موقع ويب مبني من لغتين معاً — HTML يضع العظام، CSS يضيف الملابس والألوان!

🏗️
HTML
HyperText Markup Language
يُحدد ما يوجد في الصفحة
العناوين، الفقرات، الصور، الروابط
🎨
CSS
Cascading Style Sheets
يُحدد كيف يبدو كل شيء
الألوان، الحجم، المسافات، الخطوط

💡 مثال واقعي

المنزل = HTML (الجدران، الغرف، الأبواب)
الديكور = CSS (الطلاء، الأثاث، الإضاءة)
بدون HTML لا يوجد شيء — بدون CSS كل شيء رمادي مملّ!

خصائص CSS

أهم خصائص CSS التي ستستخدمها دائماً! 🎨

background-color
لون الخلفية
